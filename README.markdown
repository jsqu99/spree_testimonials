SpreeTestimonials
=================

Testimonials (guestbook) for Spree shop.


Installation
=======

Add to Gemfile:
  ```
  gem 'spree_testimonials', github: 'jsqu99/spree_testimonials', branch: 'master'
  ```


and then run
  ```
  bundle install
  bundle exec rails g spree_testimonials:install
  bundle exec rake db:migrate
  ```
